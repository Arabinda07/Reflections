# Brand Alignment Refactor Report

## Purpose

This report documents the refactor of the strategic fundraising docs from an AI-first narrative to the brand-led Reflections narrative.

The business intent did not change. Reflections still aims to become a venture-scale personal memory platform. The narrative architecture changed so the strategy now follows the product philosophy:

People -> Thinking -> Writing -> Trust -> Memory -> Understanding -> AI -> Platform.

## framework-map.md

### Part 1: Narrative audit

Strengths:

- The original map selected the right frameworks and rejected unnecessary deck methods.
- It kept the deck short, investor-focused, and proof-oriented.

Weaknesses:

- The Pyramid/BLUF row opened with "Private AI Memory company."
- Zuora/category framing started from AI needing context.
- The user was present, but not yet the narrative protagonist.

Contradictions with brand docs:

- Brand docs say the main character is the user writing.
- AI should be optional and late.
- The old framework map treated AI as the opening category force.

### Part 2: Recommended structural changes

- Keep the same frameworks.
- Reframe each framework around the human-first chain.
- Use category language only after writing, trust, and memory are established.

## Master consistency report

### Investment narrative

The strategic docs now use one investment narrative: Reflections starts with private writing, earns trust, lets memory develop, and uses AI only after trusted context exists. The investment case remains venture-scale because trusted personal memory can expand into Life Wiki, RelationshipOS, decision trails, reminders, and AI services.

### Category definition

Primary strategic frame: Trusted Reflection Infrastructure.

Investor shorthand after context is established: Private AI Memory.

The docs no longer require investors to accept an AI category before understanding the human behavior.

### Terminology

Consistent terms across the docs:

- private writing
- trusted reflection
- trusted personal memory
- Invited AI
- Personal Memory OS
- Personal Memory Graph
- validation roadmap

Terms avoided as primary positioning:

- AI-first journal framing
- journaling app
- memory app
- wellness product

### Philosophy

The docs now match the brand philosophy: the user is the main character, writing comes first, AI is optional, trust is concrete, and memory develops over time.

### Storytelling

The deck and thesis now follow the required hierarchy:

People -> Thinking -> Writing -> Trust -> Memory -> Understanding -> AI -> Platform.

### Internal consistency

The memo, thesis, deck plan, framework map, red-team doc, and glossary now share the same spine. Differences are functional: the memo is terse, the thesis explains, the deck plans, the framework map governs method, and the red-team doc attacks weak points.

### VC coherence verdict

Yes. If a VC read every strategic document in sequence, they would see one coherent company with one coherent worldview: Reflections helps people think through private writing, turns that trust into personal memory, and uses AI only when it serves the user's understanding.
